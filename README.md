# 🛒 Nigeria Market Sales Dashboard

An interactive business intelligence dashboard analyzing
1,000 sales transactions across 26 Nigerian food products,
built with Python, Flask, and Chart.js.

## 🎮 Interactive Features

- Filter by Month, Category, or Product
- Toggle between Revenue and Quantity views
- Dynamic charts that update instantly with filters
- Sortable and searchable transaction data table
- Pagination with 15 rows per page
- Auto-generated insights based on filtered data

## 📊 Charts Included

- Monthly Revenue/Quantity trend line chart
- Revenue by Category doughnut chart
- Top 5 Products bar chart
- Dynamic summary cards

## 🛠️ Tech Stack

- **Python** — Data generation and analysis
- **Pandas** — Data processing
- **Flask** — Web framework
- **Chart.js** — Interactive charts
- **HTML/CSS/JavaScript** — Frontend dashboard

## 📁 Data Source

National Bureau of Statistics (NBS) Nigeria
Selected Food Prices Watch 2024/2025
https://nigerianstat.gov.ng

## 🚀 How to Run Locally

1. Clone the repository
2. Install dependencies:
pip install -r requirements.txt
3. Generate data first:
python generate_data.py
4. Run the app:
python app.py
5. Open browser at `http://localhost:5000`
